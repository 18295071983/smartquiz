## 2. 核心功能模块

### 2.1 AIAgentEngine - 智能代理引擎

**职责：**
- 会话管理和状态跟踪
- 任务分解和执行控制
- 上下文管理和记忆
- 多步骤推理和验证

**关键特性：**
```
- 会话持久化：支持会话恢复
- 任务队列管理：支持并行/串行任务
- 思考过程可视化：实时显示推理步骤
- 状态机管理：完整的状态流转
- 缓存机制：智能缓存减少重复请求
- 超时和重试机制：生产级可靠性
```

**核心接口：**
```java
public interface OnAgentStepListener {
    void onStepStart(String stepType, String description);
    void onStepProgress(String stepType, String description, Object data);
    void onStepComplete(String stepType, String result);
    void onThinking(String thought);
    void onAction(String action);
    void onObservation(String observation);
}
```

### 2.2 AgentAICaller - AI 调用器

**职责：**
- 封装AI调用逻辑
- 支持思考过程模式
- 管理思考步骤状态
- 流式响应处理

**关键特性：**
```
- 完整的思考步骤生命周期管理
- 流式响应支持
- 错误恢复和重试
- 超时控制
- Token 安全管理
- 请求追踪和监控
```

### 2.3 TaskDecomposer - 任务分解器

**职责：**
- 分析用户需求
- 生成执行计划
- 分解复杂任务为子任务
- 任务依赖关系管理

**关键特性：**
```
- 智能任务识别
- 依赖图构建
- 并行任务检测
- 任务优先级管理
- 执行时间估计
```

### 2.4 工具管理系统

#### AgentToolsManager - 工具管理器

**通用工具 (8个)：**
```
1. THINKING_PROCESS - 思考过程管理
2. CONTEXT_COMPRESS - 上下文压缩
3. MEMORY_RECALL - 记忆回想
4. TASK_VALIDATE - 任务验证
5. RESULT_FORMAT - 结果格式化
6. ERROR_HANDLER - 错误处理
7. CACHE_MANAGER - 缓存管理
8. SESSION_MANAGER - 会话管理
```

**AI工具 (14个)：**
```
1. GENERATE_QUESTIONS - 题目生成
2. ANALYZE_QUESTION - 题目分析
3. LEARNING_PLAN - 学习计划
4. NOTE_MANAGEMENT - 笔记管理
5. REVIEW_MANAGEMENT - 复习管理
6. WRONG_QUESTION - 错题管理
7. STATISTICS - 统计分析
8. TRANSLATE - 翻译
9. WEATHER_QUERY - 天气查询
10. WEB_SEARCH - 网络搜索
11. DB_EXPORT - 数据库导出
12. DB_IMPORT - 数据库导入
13. FILE_MANAGE - 文件管理
14. AI_CHAT - AI聊天
```

**系统工具 (5个)：**
```
1. FILE_GENERATOR - 文件生成器
2. PERMISSION_MANAGER - 权限管理
3. FILE_SYSTEM - 文件系统操作
4. SYSTEM_INFO - 系统信息
5. CLI_TOOL - CLI工具
```

### 2.5 Skill 系统 - 技能管理

**架构：**
```
Skill (技能定义)
├── SkillMetadata (元数据)
├── SkillParameters (参数定义)
└── SkillExample (示例)

SkillLoader (技能加载器)
├── 从Assets加载
├── 从文件加载
└── 从网络加载

SkillManager (技能管理器)
├── 技能注册
├── 技能查找
└── 技能状态管理

DynamicSkillExecutor (动态技能执行器)
├── 意图识别
├── 参数提取
├── 工具调用
└── 结果处理
```

## 3. 生产环境优化

**1. 缓存策略**
```
- 会话缓存：最近20个会话
- 工具调用缓存：防止重复执行
- AI响应缓存：相同输入直接返回
- TTL 管理：自动过期无效缓存
```
