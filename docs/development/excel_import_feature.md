# Excel导入功能说明文档

## 1. 功能概述

Excel导入功能是应用的核心功能之一，允许用户从Excel文件中导入题目数据到应用中。该功能支持多种题型的解析和处理，提供智能列映射和错误处理，确保数据导入的准确性和可靠性。

## 2. 支持的文件格式

- **Excel格式**：.xls、.xlsx
- **CSV格式**：.csv
- **JSON格式**：.json

## 3. 导入流程

1. **文件选择**：用户选择要导入的Excel文件
2. **工作表选择**：如果是Excel文件，用户可以选择要导入的工作表
3. **字段映射**：系统自动检测表头并进行字段映射，用户也可以手动调整
4. **题型映射**：系统检测文件中的题型并允许用户进行映射
5. **导入确认**：显示导入预览信息，包括有效题目数、无效题目数等
6. **执行导入**：开始导入数据到应用中
7. **导入完成**：显示导入结果，包括成功导入的题目数

## 4. 主要功能特性

### 4.1 智能字段映射

- 自动检测Excel表头并进行字段映射
- 支持模糊匹配，提高映射准确性
- 允许用户手动调整字段映射
- 支持保存映射配置作为模板

### 4.2 题型映射

- 自动检测文件中的题型
- 允许用户将自定义题型映射到系统支持的题型
- 支持批量映射和单个映射

### 4.3 错误处理

- 详细的错误信息，包括行号和错误原因
- 支持跳过无效题目继续导入
- 提供错误日志，便于用户排查问题

### 4.4 内存管理

- 动态调整批处理大小，根据内存情况优化导入速度
- 定期检查内存使用情况，避免内存溢出
- 支持大数据量导入，最大支持10000道题目

### 4.5 进度显示

- 实时显示导入进度
- 提供准确的进度百分比
- 显示当前处理的行数和总行数

### 4.6 导入模板

- 支持保存导入配置作为模板
- 支持使用模板快速导入相同格式的文件
- 模板管理功能，包括创建、删除和修改模板

### 4.7 导入历史记录

- 记录导入历史，最多保存50条记录
- 显示导入时间、文件名、导入题目数等信息
- 支持清空历史记录

### 4.8 导入预览

- 显示导入预览信息，包括有效题目数、无效题目数、重复题目数
- 显示字段映射情况
- 提供题型分布和难度分布统计
- 估算导入时间和内存使用情况

## 5. 使用方法

### 5.1 基本导入

1. 进入应用的导入页面
2. 点击"选择文件"按钮，选择要导入的Excel文件
3. 如果是Excel文件，选择要导入的工作表
4. 系统自动进行字段映射，用户可以根据需要调整
5. 系统检测文件中的题型，用户可以进行题型映射
6. 查看导入确认信息，确认无误后点击"开始导入"
7. 等待导入完成，查看导入结果

### 5.2 使用模板导入

1. 进入应用的导入页面
2. 点击"使用模板导入"按钮
3. 选择要使用的导入模板
4. 选择要导入的Excel文件
5. 系统根据模板自动进行字段映射和设置
6. 查看导入确认信息，确认无误后点击"开始导入"
7. 等待导入完成，查看导入结果

### 5.3 保存导入模板

1. 完成一次导入操作后，在导入结果页面点击"保存为模板"
2. 输入模板名称
3. 点击"保存"按钮
4. 模板保存成功后，可以在模板列表中查看和使用

## 6. 注意事项

1. **文件格式**：确保Excel文件格式正确，避免使用过于复杂的公式和格式
2. **表头规范**：建议使用标准的表头名称，如"题目"、"选项A"、"选项B"、"正确答案"等
3. **数据完整性**：确保数据完整，特别是必填字段（题目和正确答案）
4. **内存限制**：导入大量数据时，注意设备内存情况，建议分批导入
5. **文件大小**：Excel文件大小建议不超过50MB，避免导入过程中出现内存不足
6. **格式兼容性**：支持Excel 97-2003 (.xls)和Excel 2007及以上 (.xlsx)格式

## 7. 常见问题及解决方案

### 7.1 导入失败，提示"内存不足"

**解决方案**：
- 减少单次导入的数据量，分批导入
- 关闭其他占用内存的应用
- 检查设备内存是否充足

### 7.2 字段映射不正确

**解决方案**：
- 手动调整字段映射
- 保存映射配置作为模板，方便下次使用
- 检查Excel表头是否规范

### 7.3 题型识别错误

**解决方案**：
- 在题型映射界面手动调整题型映射
- 确保Excel中的题型名称清晰明确

### 7.4 导入速度慢

**解决方案**：
- 确保设备性能良好
- 减少单次导入的数据量
- 检查Excel文件大小，避免过大的文件

### 7.5 导入后题目显示异常

**解决方案**：
- 检查Excel文件中的数据格式
- 确保必填字段完整
- 查看导入错误日志，排查具体问题

## 8. 技术实现

Excel导入功能使用Apache POI库实现，主要包括以下组件：

- **ExcelUtil**：核心工具类，处理Excel文件的读取和解析
- **ImportActivity**：导入活动，处理用户交互和导入流程
- **SmartMappingActivity**：智能映射活动，处理字段映射和题型映射
- **MappingEditorActivity**：映射编辑活动，允许用户手动调整映射
- **QuestionTypeMapperActivity**：题型映射活动，处理题型映射

## 9. 性能优化

1. **批处理**：采用批处理方式导入数据，减少数据库操作次数
2. **内存管理**：动态调整批处理大小，根据内存情况优化导入速度
3. **并行处理**：支持并行处理，提高导入速度
4. **错误处理**：跳过无效题目继续导入，提高导入成功率
5. **缓存机制**：缓存导入配置和映射信息，提高重复导入的速度
