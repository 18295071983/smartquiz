## 1. 题库管理模块

> 核心路径：`com.smartquiz.quiz` / `com.smartquiz.question`

### 1.1 手动创建题目

| 功能项 | 说明 |
|--------|------|
| 单选题 | 支持 4~6 个选项，标记正确答案，可设置分值 |
| 多选题 | 支持 4~8 个选项，多答案标记，支持部分得分模式 |
| 判断题 | 正确/错误二元选项，快速录入 |
| 填空题 | 支持多空填空，每空独立判分 |
| 简答题 | 自由文本答案，支持关键词匹配评分 |
| 富文本题目 | 支持 Markdown 格式题干，可嵌入图片、公式（LaTeX）、代码块 |
| 题目属性 | 难度等级（1~5星）、分类标签、来源标注、备注 |

### 1.2 批量导入

| 导入格式 | 功能描述 |
|----------|----------|
| Excel (.xlsx/.xls) | 按模板格式导入，自动识别列映射，支持多 Sheet |
| CSV | 自定义分隔符，编码自动检测（UTF-8/GBK），列名自动匹配 |
| JSON | 结构化题库导入，支持嵌套分类结构 |
| Markdown | 按标题层级解析题目，支持标准 Markdown 题库格式 |

**导入流程：**

```
选择文件 → 格式检测 → 预览解析结果 → 列映射/字段匹配 → 冲突策略选择 → 导入执行 → 导入结果报告
```

**冲突策略：**

- 跳过重复（按题目文本哈希判断）
- 覆盖已有（匹配到相同 ID 或哈希时更新）
- 始终新增（保留重复）

### 1.3 导出功能

| 导出格式 | 说明 |
|----------|------|
| Excel (.xlsx) | 标准表格格式，支持样式定制 |
| CSV | 纯文本表格，兼容性强 |
| HTML | 可独立浏览的网页格式，含排版样式 |
| JSON | 结构化数据，可再次导入 |
| Markdown (.md) | 纯文本 Markdown 题库格式 |
| PDF | 排版精美的 PDF 文档 |
| Word (.docx) | 可编辑的 Word 文档 |
| 长图 (.png) | 适合分享的长截图格式，支持 24 种导出模板 |

**导出范围选项：**

- 全部题库
- 按分类导出
- 按标签导出
- 按难度导出
- 自定义筛选结果导出

### 1.4 24 种导出模板

导出模板分为以下风格类别：

| 类别 | 数量 | 风格说明 |
|------|------|----------|
| 简约风格 | 6 种 | 极简白底、灰调商务、线条简约、留白设计、微阴影、边框卡片 |
| 学术风格 | 4 种 | 试卷标准、答题卡式、教材排版、提纲式 |
| 彩色风格 | 5 种 | 渐变主题、色块分区、标签配色、品牌色、柔和色系 |
| 暗色风格 | 3 种 | 深色主题、夜间阅读、高端黑金 |
| 创意风格 | 4 种 | 手写笔记风、杂志排版、信息图风、极客终端风 |
| 复古风格 | 2 种 | 羊皮纸、旧书排版 |

每套模板包含：封面页、目录页、题目排版、答案区、统计页。

### 1.5 题目搜索、筛选与排序

**搜索功能：**

| 搜索方式 | 说明 |
|----------|------|
| 全文搜索 | 基于 SQLite FTS5 的全文索引，支持中文分词 |
| 关键词搜索 | 精确匹配题目文本、选项、标签 |
| 高级搜索 | 组合条件：分类 + 难度 + 题型 + 日期范围 + 来源 |
| 语音搜索 | 语音转文字后搜索 |
| 图片搜索 | OCR 识别后搜索 |

**筛选条件：**

- 题型筛选（单选/多选/判断/填空/简答）
- 难度筛选（1~5 星滑块）
- 分类筛选（树形分类选择器）
- 标签筛选（多标签 AND/OR 组合）
- 来源筛选
- 日期范围筛选
- 答题状态（未答/已答/答错/答对）
- 收藏状态

**排序方式：**

- 创建时间（升序/降序）
- 修改时间
- 难度等级
- 答题次数
- 错误次数
- 最近答题时间
- 自定义拖拽排序

### 1.6 分类与标签管理

**分类管理：**

- 树形层级结构，支持无限层级嵌套
- 拖拽排序分类
- 分类重命名、移动、合并
- 分类统计（含题目数量、平均难度）
- 预设分类模板（学科分类、考试分类等）

**标签管理：**

- 扁平标签系统，每个题目可关联多个标签
- 标签颜色标记
- 标签自动补全
- 常用标签快捷添加
- 标签云视图
- 标签合并与清理

## 2. 答题模块

> 核心路径：`com.smartquiz.quizmode`

### 2.1 挑战模式（Challenge Mode）

| 特性 | 说明 |
|------|------|
| 模式描述 | 逐题挑战，即时反馈对错，适合碎片化练习 |
| 题目来源 | 可按分类/标签/难度随机抽取或顺序作答 |
| 答题流程 | 显示题目 → 选择答案 → 即时判定正误 → 显示解析 → 下一题 |
| 错误处理 | 答错自动加入错题本 |
| 中断恢复 | 支持中途退出，下次可继续 |
| 连击系统 | 连续答对触发连击特效和积分加成 |
| 限时模式 | 可开启每题倒计时（10s/15s/30s/60s 可选） |
| 生命值 | 可选生命值机制（3~5 条命），答错扣命，归零结束 |

### 2.2 考试模式（Exam Mode）

| 特性 | 说明 |
|------|------|
| 模式描述 | 模拟真实考试环境，批量作答后统一判分 |
| 试卷配置 | 自定义题目数量、题型配比、难度分布、总分 |
| 计时功能 | 总时长倒计时，可暂停，超时自动交卷 |
| 答题界面 | 题号导航、已答/未答标记、标记不确定 |
| 交卷判分 | 全部答完或时间到后统一判分，生成成绩单 |
| 成绩报告 | 分数、正确率、用时统计、各题型得分率、难度分布分析 |
| 错题回顾 | 考后逐一回顾错题及解析 |
| 历史成绩 | 所有考试成绩存档，按时间/分数排序 |

### 2.3 练习模式（Practice Mode）

| 特性 | 说明 |
|------|------|
| 模式描述 | 无压力练习，可随时查看答案和解析 |
| 自由切换 | 随时跳过、查看答案、标记收藏 |
| 即时解析 | 每题可立即查看详细解析 |
| 无限制 | 无时间限制、无次数限制、无评分 |
| 进度记录 | 记录练习历史和题目完成状态 |
| 针对性练习 | 可按错题、收藏、特定知识点筛选范围 |

### 2.4 背诵模式（Recite Mode）

| 特性 | 说明 |
|------|------|
| 模式描述 | 卡片式翻转背诵，适合记忆类题目 |
| 卡片界面 | 正面显示题干，点击翻转显示答案 |
| 记忆评估 | 翻转后自评记忆程度（完全不会/模糊/熟练） |
| 间隔重复 | 基于艾宾浩斯遗忘曲线自动安排复习间隔 |
| 复习提醒 | 到期提醒复习即将遗忘的题目 |
| 进度统计 | 记忆曲线图表、掌握率统计 |
| 多种卡片样式 | 标准卡片、填空卡片、配对卡片 |

### 2.5 成绩追踪与统计

**个人统计面板：**

| 统计项 | 说明 |
|--------|------|
| 总答题数 | 历史累计答题数量 |
| 总正确率 | 整体正确率百分比 |
| 各题型正确率 | 按单选题/多选题/判断题/填空题/简答题分别统计 |
| 难度分布 | 各难度等级的正确率分布 |
| 答题趋势图 | 日/周/月维度的正确率变化曲线 |
| 用时统计 | 平均每题用时、总学习时长 |
| 连胜记录 | 最高连胜次数、当前连胜 |
| 积分排名 | 积分累积与排行榜（可选） |
| 学习热力图 | 类似 GitHub 贡献图的每日学习量可视化 |

### 2.6 错题本

| 功能 | 说明 |
|------|------|
| 自动收录 | 挑战模式、考试模式中答错的题目自动加入 |
| 手动添加 | 可手动将题目加入错题本 |
| 错题复习 | 针对错题本专项练习 |
| 错题移除 | 连续答对 N 次后自动移出（可配置） |
| 错题统计 | 错误次数、最后错误时间、错误类型标记 |
| 导出错题 | 将错题导出为独立题库文件 |

### 2.7 计时模式

| 模式 | 说明 |
|------|------|
| 不限时 | 无时间限制，自由作答 |
| 每题限时 | 每道题独立倒计时，超时判错或跳过 |
| 总时限 | 整场考试/练习的总时长限制 |
| 番茄钟 | 25 分钟专注 + 5 分钟休息循环 |
| 自定义 | 自由设置时长参数 |

## 3. AI 智能模块

> 核心路径：`com.smartquiz.ai`

### 3.1 Agent 智能体系统

| 特性 | 说明 |
|------|------|
| 多引擎架构 | 支持模块化引擎注册，按任务类型自动路由 |
| 任务分解 | 复杂任务自动拆分为子任务，并行或串行执行 |
| 上下文管理 | 会话上下文保持，支持多轮对话记忆 |
| 引擎调度 | 按任务优先级和引擎负载动态调度 |
| 状态追踪 | 任务执行状态实时追踪与反馈 |
| 错误恢复 | 子任务失败自动重试或降级处理 |

**引擎列表：**

| 引擎名称 | 用途 |
|----------|------|
| DeepThinkingEngine | 深度推理分析，复杂题目解答 |
| CreativeWritingEngine | 创意写作，题目创意生成 |
| IntentRecognitionEngine | 用户意图智能识别与路由 |
| QuestionAnalysisEngine | 题目解析、知识点提取 |
| KnowledgeRetrievalEngine | 知识库检索增强（RAG） |
| CodeInterpreterEngine | 代码执行与解释 |

### 3.2 AI Chat 对话

| 特性 | 说明 |
|------|------|
| 流式输出 | 基于 SSE/WebSocket 的逐字流式输出，打字机效果 |
| 多轮对话 | 完整上下文保持，历史消息管理 |
| 消息类型 | 文本、图片（多模态）、代码块、公式、表格 |
| 对话管理 | 新建/删除/重命名对话，对话列表搜索 |
| 快捷指令 | 预设常用提问模板（"解析此题""生成相似题"等） |

| Markdown 渲染 | 完整 Markdown 支持，含 LaTeX 数学公式、代码高亮、流程图 |

### 3.3 深度思考引擎（Deep Thinking Engine）

| 特性 | 说明 |
|------|------|
| 思维链推理 | Chain-of-Thought 分步推理展示 |
| 推理过程可视化 | 展示 AI 的分析步骤、中间结论 |
| 多角度分析 | 从不同角度审视问题，提供多种解法 |
| 知识溯源 | 标注推理依据和知识来源 |

### 3.4 创意写作引擎（Creative Writing Engine）

|------|------|

| 文案润色 | 题目文本的语法纠正和表达优化 |

### 3.10 四大 Feature 功能模块

#### 3.10.1 学习助手（LearningAssistant）

| 功能 | 说明 |
|------|------|
| 智能答疑 | 拍照/输入题目，AI 逐步讲解 |
| 知识点梳理 | 根据学习进度自动整理知识图谱 |
| 薄弱点分析 | AI 分析答题数据，识别薄弱知识点 |
