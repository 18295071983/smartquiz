# SmartQuiz HTML5 网页应用

这是一个基于HTML5的SmartQuiz网页应用，无需服务器即可在本地运行。

## 功能特点

- **学习管理**：题目管理、开始测验、学习计划、错题集
- **工具**：OCR识别、AI助手、导入文件、数据导出、学习笔记、数据备份
- **设置**：用户管理、主题设置、语言设置
- **测试**：环境检测、测试按钮

## 如何使用

1. **直接打开**：在浏览器中直接打开 `index.html` 文件
2. **导航测试**：打开 `test-navigation.html` 文件可以快速测试所有页面的导航功能

## 技术实现

- **前端技术**：HTML5、CSS3、JavaScript
- **响应式设计**：适配不同屏幕尺寸
- **模块化架构**：按功能模块化组织代码
- **本地存储**：使用localStorage存储数据
- **API模拟**：内置模拟数据，无需后端服务器
- **主题支持**：支持浅色、深色、蓝色、绿色主题
- **多语言支持**：支持简体中文、英语、繁体中文等多种语言

## 注意事项

1. **本地运行**：所有页面都可以在本地直接打开，无需服务器
2. **数据存储**：使用localStorage存储数据，刷新页面后数据会保留
3. **文件上传**：OCR和导入功能支持本地文件上传
4. **文件下载**：导出和备份功能支持本地文件下载
5. **浏览器兼容性**：建议使用现代浏览器如Chrome、Firefox、Edge等

## 功能说明

### 学习管理
- **题目管理**：查看、搜索、筛选题目
- **开始测验**：进行测验，计时，查看结果
- **学习计划**：创建、编辑、跟踪学习计划
- **错题集**：查看、复习、管理错题

### 工具
- **OCR识别**：上传图片识别文字
- **AI助手**：与AI助手聊天，获取学习建议
- **导入文件**：导入CSV、Excel、JSON格式的题目数据
- **数据导出**：导出数据为CSV、Excel、JSON格式
- **学习笔记**：创建、编辑、管理学习笔记
- **数据备份**：创建、恢复备份，设置自动备份

### 设置
- **用户管理**：编辑个人信息，修改密码，管理通知设置
- **主题设置**：选择主题，调整字体和界面设置
- **语言设置**：选择界面语言，设置翻译偏好

### 测试
- **环境检测**：检测浏览器、设备、网络和功能支持
- **测试按钮**：测试功能、UI、性能和错误处理

## 本地运行方法

1. 下载或克隆项目到本地
2. 在文件浏览器中找到 `index.html` 文件
3. 双击 `index.html` 文件，在浏览器中打开
4. 开始使用SmartQuiz应用

## 浏览器支持

- Chrome (推荐)
- Firefox
- Edge
- Safari
